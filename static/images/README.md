### Image Specifications
- **Recommended Size**: 1200x800px for hero images
- **Format**: PNG with transparent background for logos/avatars
- **Format**: JPG for photographs
- **Compression**: Optimized for web (under 200KB per image)

### Image Categories

#### Hospital & Medical
- Professional medical facility photography
- Clean, modern hospital environments
- Medical equipment and technology
- Staff in professional attire

#### Donor Related
- Diverse group of donors
- Blood donation process
- Community engagement
- Success stories and testimonials

#### Emergency & Critical Care
- Emergency medical response
- Critical care scenarios
- Life-saving moments
- Medical urgency

### Color Scheme
- **Primary**: Blood red (#ef4444)
- **Secondary**: Medical green (#10b981)
- **Accent**: Professional blues and grays
- **Background**: Clean whites and light grays

### Usage Guidelines
1. Use high-quality, professional images
2. Ensure diversity and inclusion in donor imagery
3. Maintain consistent color grading
4. Optimize for fast loading
5. Use appropriate alt text for accessibility

### Placeholder Images
Until actual images are available, the system will use:
- CSS gradients for hero sections
- SVG patterns for backgrounds
- Icon-based placeholders for missing images

### Image Optimization
- Use WebP format when possible
- Implement lazy loading for below-fold images
- Serve responsive images based on device
- Use CDNs for faster delivery
